AutoGeo — Automatic Geometrization & Squaring for Illustrator

AutoGeo is an open-source plugin for Adobe Illustrator designed to automate geometrization, shape regularization, and precision squaring of vector artwork.

Built for designers, illustrators, and art directors, it transforms freeform paths into clean geometric constructions, aligning curves, angles, proportions, and axes according to configurable rules. The plugin helps convert rough sketches, hand-drawn vectors, or irregular shapes into structured, rational, production-ready geometry.

Features
Automatic geometrization of vector shapes
Shape squaring and orthogonal alignment
Curve simplification and rationalization
Smart angle snapping and proportional correction
Grid / modular system fitting
Non-destructive workflow
Open-source and extensible architecture
Use cases

AutoGeo is ideal for:

logo refinement
icon design
lettering cleanup
technical illustration
brand system construction
vector optimization for production
